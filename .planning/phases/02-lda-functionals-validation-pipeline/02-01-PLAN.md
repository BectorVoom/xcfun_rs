**helpers.rs:** Implement these shared functions, all generic over `T: Num`. Port algorithms character-for-character from C++ sources:

1. `pub fn ufunc<T: Num>(x: &T, a: f64) -> T` -- spin interpolation from C++ `specmath.hpp`. Formula: `(1+x)^a + (1-x)^a`. Use `T::one()` and `T::from_f64()` for constants, `.pow(a)` for exponentiation.

2. `pub fn vwn_f<T: Num>(s: &T, p: &[f64; 4]) -> T` -- VWN parameterized function from C++ `vwn.hpp`. Parameters `p = [x0, A, b, c]`. Pre-compute all f64 constants from parameters OUTSIDE the generic path (as plain f64 arithmetic), then inject via `T::from_f64()`. The formula is:
   `0.5 * A * (2*ln(s) + a*ln(X(s)) - b_coeff*ln(Y(s)) + c_coeff*atan(Z(s)))`
   where X(s) = s^2 + b*s + c, Y(s) = s - x0, Z(s) = disc / (2*s + b), disc = sqrt(4*c - b^2),
   a = x0*b/(x0^2 + x0*b + c) - 1, b_coeff = 2*(same ratio - 1) + 2,
   c_coeff = 2*b * (1/disc - x0/(X0 * disc/(b+2*x0))).
   CRITICAL: Copy the exact constant derivation logic from vwn.hpp lines defining `a`, `b`, `c` in the `vwn_f()` function. Match the C++ formula structure exactly.

3. `pub fn pw92_eopt<T: Num>(sqrt_r: &T, t: &[f64; 7]) -> T` -- PW92 parameterized function from `pw92eps.hpp`. Formula: `-2*t[0] * (1 + t[1]*sqrt_r^2) * ln(1 + 0.5/(t[0] * sqrt_r * (t[2] + sqrt_r*(t[3] + sqrt_r*(t[4] + t[5]*sqrt_r)))))`. Use polynomial nesting for the denominator.

4. `pub fn pw92_omega<T: Num>(zeta: &T) -> T` -- spin interpolation from `pw92eps.hpp`. Uses `ufunc(zeta, 4.0/3.0)`. Denominator is `2.0 * 2.0_f64.powf(1.0/3.0) - 2.0` computed as f64 constant (NOT the `XCFUN_REF_PW92C` hardcoded 0.5198421 -- per pitfall 7 in RESEARCH.md).

5. `pub fn pz81_eld<T: Num>(x: &T, cb: &[f64; 3]) -> T` -- PZ81 low-density from `pz81c.hpp`: `cb[0] / (1 + cb[1]*sqrt(x) + cb[2]*x)`.

6. `pub fn pz81_ehd<T: Num>(x: &T, c: &[f64; 4]) -> T` -- PZ81 high-density from `pz81c.hpp`: `c[0] + ln(x) * (c[1] + x * c[3]) + c[2] * x`. Match the C++ indexing exactly.

7. PZ81 and PW92 parameter constants as `pub const` arrays. Copy values character-for-character from C++ sources:
   - PZ81_C: 4 arrays of [f64; 3] or [f64; 4] (see pz81c.hpp)
   - PW92_TUVWXYP: 3 arrays of [f64; 7] (see pw92eps.hpp)
   - VWN3_PARA, VWN3_FERRO: [f64; 4] arrays
   - VWN5_PARA, VWN5_FERRO, VWN5_INTER: [f64; 4] arrays

IMPORTANT: Use the DEFAULT (non-`XCFUN_VWN5_REF`, non-`XCFUN_REF_PW92C`) parameter values from C++. Copy parameter values character-for-character -- even minor rounding causes errors exceeding 1e-12.
